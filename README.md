User einer Sicherheitsgruppe hinzufügen - interaktiv

Dieses Skript ist interaktiv und soll das hinzufügen von Usern vereinfachen.
Oft erhält man nur Vornamen und Nachnamen. Das Skript sucht anhand der Eingabe von Namen und fügt sie hinzu. 
Bei mehreren Treffern, Auswahl möglich.



 Adding Users to a Security Group - Interactive This script is interactive and is intended to make it easier to add users. 
 Often you only get first names and last names. The script searches for and adds names by typing names. In case of multiple hits, selection possible.
